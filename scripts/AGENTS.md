# Instrucciones para Asistentes IA

## Regla: Notificar descargas

Cuando ejecutes cualquier script que descargue música (Tidal, Qobuz, YouTube, etc.) y un track se descargue exitosamente, DEBES notificar al usuario con:

> Se descargó: {ArtistName} — {TrackTitle}

Esto aplica a todos los scripts en este repositorio, incluyendo `descargar_artistas.py`, `descargar_links_sueltos.py`, scripts en `scratch/`, etc.
